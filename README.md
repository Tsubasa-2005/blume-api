# blume-api

blumeコードの生成、読み取り機能を提供するAPIです

blumeコードは花の形をした任意のデータを埋め込める二次元コードです

## 開発背景

ハックツハッカソンステゴカップ提出作品

ハッカソンテーマ「〇〇の魔法」

私たちのテーマは花畑を出す魔法です
